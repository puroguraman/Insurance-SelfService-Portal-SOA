🛡️ MyInsurance Portal - SOA & Digital Transformation

📝 Project Overview:This project was developed as part of the Master's Program "Information Systems & Services" at the University of Piraeus. It addresses the digital transformation of insurance services by implementing a Self-Service Web Portal. The system allows policyholders to manage their portfolios, modify coverages (Underwriting), and execute payments in real-time. 

🚀 Key FeaturesPortfolio Overview: Categorized view of policies (Auto, Health, Life, Home). 
Dynamic Underwriting: Real-time calculation of premiums when adding/removing coverages. 

-Online Payments: Integrated payment gateway simulation for immediate policy renewal

-SSM Analysis: Problem structured using Soft Systems Methodology (Rich Pictures, CATWOE)

-Process Orchestration: Business logic separation using Bonita BPM

🛠️ Technical SpecificationsBackend: Java 17, Spring Boot, Spring Security (RBAC). 

-Database: MySQL with Audit Logging for non-repudiation. 

-Architecture: Service-Oriented Architecture (SOA) & DevOps approach. 

-Containerization: Docker & Docker Compose for consistent deployment. 

-Testing: Automated Unit Testing with JUnit 5.
